 # 自动生成艾宾浩斯遗忘曲线的Excel
## 功能介绍
根据你的任务安排以及任务的开始日期自动生成每天对应的任务号
## 使用方法
运行gen_Ebbinghaus.py，按照提示输入即可将会生成excel文件在项目文件夹
## 环境要求
环境：`python3`
依赖包：`pip install openpyxl`
